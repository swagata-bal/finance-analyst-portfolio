## LBO / M&A Modelling

This folder contains a full Leveraged Buyout (LBO) model built for a cash-generative company.

Key components:
- Sources & Uses
- Multi-tranche debt structure
- Debt repayment and cash sweep
- IRR and MOIC calculation
- Sensitivity and downside scenarios

An investment memo accompanies the model to explain deal rationale, value creation, risks, and exit strategy.
